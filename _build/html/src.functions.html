src.functions.printing_and_logging.print_and_log(text, colour)[source]

Print the given text with the specified color and log it accordingly.

Steps: 1. Convert the input text to a string. 2. If a color is specified:

  1. Print the text with the specified color using the Fore module from colorama.

  2. If the color is ‘RED’:

    1. Log the text as an error using logging.error().

    2. Try to post the error to an API using api_post_string().

    3. If an exception occurs during the API post, log the error using logging.error().

  3. Otherwise, log the text as information using logging.info().

  1. If no color is specified:

    1. Print the text without any color.

    2. Log the text as information using logging.info().

Parameters: - text (str): The text to be printed and logged. - colour (str): The color to be used for printing the text. Can be ‘RED’ or any other color supported by the Fore module from colorama.

Returns: None
